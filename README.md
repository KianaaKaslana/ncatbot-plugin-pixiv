# Pixiv 插件

这是一个为 NcatBot 设计的 Pixiv 插件，用于获取 P 站推荐图片并发送到群聊。

## 功能

- 通过命令获取 P 站推荐的图片
- 支持获取插画和多页插画
- 可以在群聊中发送推荐图片的合并转发消息

## 依赖

- ncatbot
- pixivpy3

## 配置

在使用前需要配置 Pixiv API 的刷新令牌：

1. 设置环境变量 `PIXIV_REFRESH_TOKEN` 为您的 Pixiv 刷新令牌
2. 如需要，配置代理设置（默认使用 Clash HTTP 代理）

## 使用方法

在群聊中发送命令 `好图推荐` 即可获取 3 张 P 站推荐图片。

## 注意事项

- 需要有效的 Pixiv API 刷新令牌
- 请确保刷新令牌的安全性，不要将其暴露在公开的代码库中
- 如在防火区域使用，需要配置代理
- 插件会将下载的图片保存在 `pixiv_download` 目录中
